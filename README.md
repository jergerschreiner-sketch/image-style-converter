# 图片插画风转换器

这是一个本地预览版。直接打开 `index.html`，上传照片后就可以生成插画风、漫画风或线稿淡彩风图像，并下载 PNG。

## 当前版本能做什么

- 保留上传照片本身的色彩基调。
- 将照片简化成更平面的色块。
- 自动提取边缘，并根据照片色调生成深色描边、细节线和浅色高光线。
- 默认保留完整画幅，不裁切成贴纸。
- 可选纸张白边、漫画框、胶片感等画面装饰。
- 支持调节色块强度、描边密度、线条粗细、精修度、装饰留白。

## 调节建议

- 想要更精致清楚：提高“精修度”，线条粗细保持 1-2。
- 想要更像插画：提高“色块强度”，适当降低“描边密度”。
- 想要更像漫画：选择“漫画高反差”，提高描边密度。
- 想要更轻盈：选择“线稿淡彩”，降低线条粗细。
- 照片细节太杂：降低“描边密度”，避免线条太碎。

## 画面装饰建议

- 无装饰：最适合做普通图片风格转换，完整保留原图画幅。
- 纸张白边：适合明信片、插画展示、作品集页面。
- 漫画框：适合高对比漫画风，画面更有分镜感。
- 胶片感：适合夕阳、夜景、街景照片。

## 使用方式

1. 双击打开 `index.html`。
2. 点击“选择一张照片”上传图片。
3. 调整风格和滑杆。
4. 点击“下载 PNG”保存结果。

## 后续方向

这个工具现在更适合做“照片到插画/漫画”的本地转换。后续可以继续增强线稿质量、增加网点/漫画阴影、加入前后对比拖拽，以及批量处理一组照片。
